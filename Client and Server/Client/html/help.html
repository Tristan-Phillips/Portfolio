Help

  • This application is made for demo as a prototype
    and does not contain full funtionality
  • Values must be enetered into the respective spin boxes
    for the buttons to become enabled
  • When containers are added to the unallocated container list
    they can be moved to a pallet represented by the number displayed
    in its respective spin box
  • After the desired pallets are allocated with containers
    They can be sent to the companion server
    provided the server is actively listening for data
